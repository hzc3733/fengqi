# 《你是你吃出来的》——读书总结

> 作者：夏萌
> 安贞医院首任临床营养科主任
> 来源：10万+临床病例 + 10年实践

---

## 一句话核心观点

**慢病的本质是细胞损伤速度超过修复速度，而修复细胞的唯一原料是食物中的七大营养素。吃药治标，改变饮食才是治本。**

---

## 作者背景与写作初衷

夏萌原是天坛医院神经内科医生，2000年患肾病（35个肾小球中7个硬化），中西医治疗三年无效。2004年偶然接触营养学，3个月调整饮食后康复，从此成为安贞医院首任临床营养科主任。

10年间诊治约10万患者，涵盖脑卒中、糖尿病、高血压、帕金森、老年痴呆等慢病，总结出**西医+营养学结合**的新诊疗模式。

## 核心框架

### 1. 疾病的本质

> 生病 = 细胞损伤速度 > 细胞修复速度

| 因素 | 对细胞的影响 |
|------|-------------|
| 遗传、外伤、病毒 | 不可控因素 |
| 饮食、运动、睡眠、吸烟、饮酒、心态 | **可控因素** |

慢病（冠心病、高血压、糖尿病、肿瘤等）都属于**生活方式病**，根源是饮食不平衡。

### 2. 七大营养素

| 营养素 | 主要功能 | 食物来源 |
|--------|---------|---------|
| 碳水化合物 | 供能（主要来源） | 米面、薯类、水果 |
| 蛋白质 | 构建细胞结构 | 肉、蛋、奶、鱼 |
| 脂类 | 细胞膜、激素、储能 | 动物油、植物油、坚果 |
| 维生素 | 辅酶，调节代谢 | 蔬菜、水果、动物内脏 |
| 矿物质 | 酶的组成、渗透压平衡 | 肉类、奶、海产品 |
| 膳食纤维 | 喂养肠道菌群 | 粗粮、蔬菜 |
| 水 | 溶剂、反应介质 | 饮用水 |

**营养失衡的典型表现：**
- 蛋白质缺乏 → 贫血、水肿、骨质疏松、频繁感冒
- 碳水过多 → 腹部肥胖、血糖高
- 脂肪不足 → 皮肤干燥、脂溶性维生素缺乏
- 维生素缺乏 → 代谢紊乱、免疫力下降

### 3. 能量平衡

**计算公式（以轻体力劳动者为例）：**

```
标准体重 = 身高(cm) - 105
每日总能量 = 标准体重 × 30千卡
```

**三大产能营养素比例（建议）：**

| 营养素 | 标准比例 | 健康优化比例 |
|--------|---------|-------------|
| 碳水化合物 | 55%~65% | 取低值 55% |
| 蛋白质 | 10%~15% | 取高值 15%~20% |
| 脂类 | 20%~30% | 取高值 30%+ |

> 注意：体重正常不等于健康！可能是脂肪超标 + 肌肉不足（隐性肥胖）。

## 人类饮食进化史（对照健康标准）

| 时代 | 饮食特点 | 健康影响 |
|------|---------|---------|
| 旧石器时代（250万年前） | 荤素搭配，食物多样 | 脑容量激增，体格健壮 |
| 农牧时代（1万年前） | 食谱变窄，谷物为主 | 进化停滞，体质衰退 |
| 工业革命（200年前） | 精加工、食品添加剂 | 营养素贬值，慢病流行 |

**结论：现代人需要回归多样化饮食，多吃肉蛋奶鱼，少吃精米精面。**

## 实践方法

### 每日饮食配比

> **35% 动物类食物 + 65% 植物类食物**

**具体执行：**
- **蛋白质**：每天 1 个鸡蛋 + 300ml 牛奶 + 150g 瘦肉/鱼
- **碳水**：粗粮占一半，米面占一半，每天至少 150g 粮食
- **脂类**：植物油 + 动物油各一半，多吃鱼虾
- **蔬菜水果**：蔬菜为主，水果限量
- **水**：及时补充，不要等渴了才喝

### 常见病饮食调理

| 疾病 | 关键饮食调整 |
|------|------------|
| 高血压 | 戒面条和咸菜，增加肉类、牛奶 |
| 糖尿病 | 戒粥，多吃瘦肉，粗粮替代精米面 |
| 帕金森 | 用食物给大脑补充营养，病情稳定 |
| 肾病 | 优质蛋白不能少，但要根据肾功能调整量 |
| 贫血 | 多吃肝脏、鸭血、红肉、鸡蛋、牛奶 |

### 因人而异原则

- 体力劳动者 → 多吃主食
- 脑力劳动者 → 多吃蛋白和脂类
- 出汗多（南方/夏天）→ 补充盐分
- 运动量大 → 能量消耗大，需相应增加所有营养

## 核心误区

| 误区 | 正确认知 |
|------|---------|
| "清淡饮食"就是少吃肉 | 清淡是指少油少盐，不是无蛋白 |
| 糖尿病不能吃肉 | 混合性食物加肉类可降低血糖升速 |
| 骨质疏松只是缺钙 | 更是缺蛋白质（骨胶原） |
| 降压药不能停 | 饮食调整到位可停药 |
| 喝粥养人 | 老粥是粗粮，现代粥（白米）是高升糖 |
| 豆浆能代替牛奶 | 豆浆缺少维生素B12和钙 |
| 植物蛋白更安全 | 动物蛋白生物利用率更高，更难吃过量 |

---

## 关键启示

1. **慢病可以预防和治愈**——在细胞损伤还没到无可挽回之前，改变生活方式永远不晚
2. **食物是最好的药物**——把厨房当成药房，把食物当成保健品
3. **没有统一食谱**——因人、因时、因地而异
4. **坚持是关键**——知道道理的人很多，真正做到的人很少
5. **医者食先**——医生如果学会营养学，将如虎添翼
